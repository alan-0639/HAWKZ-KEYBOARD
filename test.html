VitePress 自定义侧边栏演示

本示例展示如何为首页的不同操作显示不同的侧边栏内容

实现原理

在 VitePress 中，可以通过配置文件动态控制侧边栏内容。以下是如何实现的：

// .vitepress/config.js export default { title: '我的文档', description: 'VitePress 自定义侧边栏示例', themeConfig: { sidebar: { '/': getSidebar() } } } // 根据当前路由或参数返回不同的侧边栏配置 function getSidebar() { // 获取当前操作类型（从URL参数或路由状态） const action = getCurrentAction(); if (action === 'action-a') { return [ { text: '操作 A 指南', items: [ { text: '开始使用', link: '/action-a/getting-started' }, { text: '基础配置', link: '/action-a/basic-config' }, { text: '高级功能', link: '/action-a/advanced' }, { text: '常见问题', link: '/action-a/faq' } ] }, // 更多侧边栏项... ] } else if (action === 'action-b') { return [ { text: '操作 B 指南', items: [ { text: '快速入门', link: '/action-b/quick-start' }, { text: '核心概念', link: '/action-b/concepts' }, { text: '配置选项', link: '/action-b/configuration' }, { text: 'API 参考', link: '/action-b/api' } ] }, // 更多侧边栏项... ] } }

通过这种方式，您可以根据用户的操作或页面状态动态切换侧边栏内容。

实际应用场景

这种技术适用于以下场景：

  • 为不同用户角色显示不同的导航菜单
  • 根据产品功能模块切换文档结构
  • 为多语言文档提供不同的侧边栏
  • 根据用户权限显示或隐藏某些文档部分
